## 🛠️ **Installation & Setup**

### 📋 **Prerequisites**
- **Python 3.8+** - [Download from python.org](https://python.org)
- **Windows 10/11** - (Current setup optimized for Windows)
- **Internet Connection** - For package installation and AI services

### ⚡ **Automated Setup**
1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/jarvis-enhanced-edition.git
   cd jarvis-enhanced-edition
   ```

2. **Run the setup script**:
   ```bash
   setup_jarvis_environment.bat
   ```

3. **Configure API keys**:
   - Edit the `.env` file created during setup
   - Add your Google AI API key and LiveKit credentials

4. **Launch Jarvis**:
   ```bash
   run_jarvis.bat
   ```

### 🔧 **Manual Setup**
```bash
# Create virtual environment
python -m venv jarvis_env
jarvis_env\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Create configuration file
copy .env.example .env
# Edit .env with your API keys

# Run Jarvis
python agent.py
```
